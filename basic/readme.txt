These are instructions for how to set up and run the flask web server

    git clone https://github.com/axo318/FlaskTutorial.git

   1) Make a new virtualenv using: virtualenv 'name' --python=python3
   2) Enable the environment with: source 'name'/bin/activate
   3) Install flask with: pip3 install flask
   4) Open the main.py file and set the PORT variable to the correct port number for testing the website
   5) Start the server using: python3 main.py
